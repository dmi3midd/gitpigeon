# Project gitpigeon

Gitpigeon - API that allows users to subscribe to email notifications about new releases of a GitHub repository

## Getting Started

*в файлі [DEVELOPMENT.md](DEVELOPMENT.md) описано хід розробки*

### 1. Клонувати репо
git clone [<url>](https://github.com/dmi3midd/gitpigeon) && cd gitpigeon
### 2. Створити .env з шаблону і заповнити свої значення
cp .env.example .env
### 3. Запустити
docker compose up --build
